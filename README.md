## Project Overview

Wasayt Media Player is a comprehensive desktop multimedia application developed using Java. The system provides a robust platform for managing, organizing, and playing various audio and video file formats. It features an intuitive user interface, playlist management capabilities, user authentication, and persistent data storage using a relational database.

This project was developed in accordance with software engineering best practices, including requirements analysis, system design using UML diagrams, implementation following object-oriented principles, and thorough testing.

## Key Features

### 1. Media Playback
- Support for multiple audio formats (MP3, WAV, AAC, OGG)
- Support for multiple video formats (MP4, AVI, MKV, FLV)
- Playback controls (Play, Pause, Stop, Next, Previous, Seek)
- Volume control and mute functionality
- Full-screen mode for video playback

### 2. Library Management
- Add media files from local storage
- Automatic metadata extraction (title, artist, album, duration)
- Categorize files by type (Audio/Video)
- Search and filter functionality
- Delete and organize media files

### 3. Playlist Management
- Create custom playlists
- Add/remove media from playlists
- Edit playlist details
- Shuffle and repeat modes
- Save playlists to database

### 4. User Management
- User registration and authentication
- Secure login system with password encryption
- User profile management
- Role-based access control

### 5. Database Integration
- Persistent storage for users, media files, and playlists
- Efficient data retrieval and updates
- Data integrity and validation

### 6. User Interface
- Modern and intuitive GUI design
- Responsive layout
- Dark/Light theme options
- Keyboard shortcuts for common operations
